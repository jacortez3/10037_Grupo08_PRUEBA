Métodos jQuery AJAX

Método / Propiedad Descripción Ejemplos
$.ajax() Realiza una solicitud AJAX asíncrona Ejemplo
$.ajaxSetup() Establece los valores predeterminados para futuras solicitudes de AJAX Ejemplo
$.get() Carga datos de un servidor usando una solicitud AJAX HTTP GET Ejemplo
$.getJSON() Carga datos codificados en JSON desde un servidor mediante una solicitud HTTP GET Ejemplo
$.getScript() Carga (y ejecuta) un JavaScript desde un servidor utilizando una solicitud AJAX HTTP GET Ejemplo
$.param() Crea una representación serializada de una matriz u objeto (se puede usar como cadena de consulta de URL para solicitudes AJAX) Ejemplo
$.post() Carga datos de un servidor usando una solicitud AJAX HTTP POST Ejemplo
ajaxComplete() Especifica una función para ejecutar cuando se completa la solicitud AJAX Ejemplo
ajaxError() Especifica una función para ejecutar cuando la solicitud AJAX se completa con un error Ejemplo
ajaxSend() Especifica una función para ejecutar antes de que se envíe la solicitud AJAX Ejemplo
ajaxStart() Especifica una función para ejecutar cuando comienza la primera solicitud de AJAX Ejemplo
ajaxStop() Especifica una función para ejecutar cuando se hayan completado todas las solicitudes de AJAX Ejemplo
ajaxSuccess() Especifica una función para ejecutar cuando una solicitud AJAX se completa con éxito Ejemplo
load() Carga datos de un servidor y coloca los datos devueltos en el elemento seleccionado Ejemplo
serialize() Codifica un conjunto de elementos de formulario como una cadena para su envío Ejemplo
serializeArray() Codifica un conjunto de elementos de formulario como una matriz de nombres y valores Ejemplo
