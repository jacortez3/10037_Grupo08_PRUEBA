Métodos de cruce de jQuery

Método / Propiedad Descripción Ejemplos
add() Agrega elementos al conjunto de elementos coincidentes Ejemplo
children() Devuelve todos los hijos directos del elemento seleccionado. Ejemplo
closest() Devuelve el primer ancestro del elemento seleccionado Ejemplo
contents() Devuelve todos los elementos secundarios directos del elemento seleccionado (incluidos los nodos de texto y comentarios) Ejemplo
each() Ejecuta una función para cada elemento coincidente Ejemplo
eq() Devuelve un elemento con un número de índice específico de los elementos seleccionados Ejemplo
filter() Reduzca el conjunto de elementos coincidentes a aquellos que coincidan con el selector o pasen la prueba de la función Ejemplo
find() Devuelve elementos descendientes del elemento seleccionado. Ejemplo
first() Devuelve el primer elemento de los elementos seleccionados. Ejemplo
has() Devuelve todos los elementos que tienen uno o más elementos dentro de ellos. Ejemplo
is() Comprueba el conjunto de elementos coincidentes con un selector/elemento/objeto jQuery y devuelve verdadero si al menos uno de estos elementos coincide con los argumentos dados Ejemplo
last() Devuelve el último elemento de los elementos seleccionados. Ejemplo
next() Devuelve el siguiente elemento hermano del elemento seleccionado Ejemplo
nextAll() Devuelve todos los siguientes elementos hermanos del elemento seleccionado Ejemplo
nextUntil() Devuelve todos los siguientes elementos hermanos entre dos argumentos dados Ejemplo
not() Devuelve elementos que no coinciden con un determinado criterio Ejemplo
offsetParent() Devuelve el primer elemento padre posicionado Ejemplo
parent() Devuelve el elemento padre directo del elemento seleccionado Ejemplo
parents() Devuelve todos los elementos antecesores del elemento seleccionado Ejemplo
parentsUntil() Devuelve todos los elementos antecesores entre dos argumentos dados Ejemplo
prev() Devuelve el elemento hermano anterior del elemento seleccionado Ejemplo
prevAll() Devuelve todos los elementos hermanos anteriores del elemento seleccionado Ejemplo
prevUntil() Devuelve todos los elementos hermanos anteriores entre dos argumentos dados Ejemplo
siblings() Devuelve todos los elementos hermanos del elemento seleccionado. Ejemplo
slice() Reduce el conjunto de elementos coincidentes a un subconjunto especificado por un rango de índices Ejemplo
