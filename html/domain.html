Método / Propiedad Descripción Ejemplos
blur() Adjunta/Activa el evento de desenfoque Ejemplo
change() Adjunta/Activa el evento de cambio Ejemplo
click() Adjunta/Activa el evento de clic Ejemplo
dblclick() Adjunta/Activa el evento de doble clic Ejemplo
event.currentTarget El elemento DOM actual dentro de la fase de difusión del evento Ejemplo
event.data Contiene los datos opcionales pasados a un método de evento cuando el controlador de ejecución actual está vinculado Ejemplo
event.delegateTarget Devuelve el elemento donde se adjuntó el controlador de eventos jQuery llamado actualmente Ejemplo
event.isDefaultPrevented() Devuelve si se llamó a event.preventDefault() para el objeto de evento Ejemplo
event.isImmediatePropagationStopped() Devuelve si se llamó a event.stopImmediatePropagation() para el objeto de evento Ejemplo
event.isPropagationStopped() Devuelve si se llamó a event.stopPropagation() para el objeto de evento Ejemplo
event.namespace Devuelve el espacio de nombres especificado cuando se activó el evento. Ejemplo
event.pageX Devuelve la posición del ratón en relación con el borde izquierdo del documento Ejemplo
event.pageY Devuelve la posición del mouse en relación con el borde superior del documento Ejemplo
event.preventDefault() Impide la acción predeterminada del evento. Ejemplo
event.relatedTarget Devuelve qué elemento se ingresa o se sale con el movimiento del mouse Ejemplo
event.result Contiene el último/anterior valor devuelto por un controlador de eventos desencadenado por el evento especificado Ejemplo
event.stopImmediatePropagation() Evita que se llame a otros controladores de eventos Ejemplo
event.stopPropagation() Evita que el evento se expanda en el árbol DOM, lo que impide que se notifique el evento a los controladores principales. Ejemplo
event.target Devuelve qué elemento DOM activó el evento Ejemplo
event.timeStamp Devuelve el número de milisegundos desde el 1 de enero de 1970, cuando se activa el evento Ejemplo
event.type Devuelve qué tipo de evento se activó Ejemplo
event.which Devuelve qué tecla del teclado o botón del mouse se presionó para el evento Ejemplo
focus() Adjunta/Activa el evento de enfoque Ejemplo
focusin() Adjunta un controlador de eventos al evento focusin Ejemplo
focusout() Adjunta un controlador de eventos al evento de enfoque Ejemplo
hover() Adjunta dos controladores de eventos al evento flotante Ejemplo
keydown() Adjunta/Activa el evento keydown Ejemplo
keypress() Adjunta/Activa el evento de pulsación de tecla Ejemplo
keyup() Adjunta/Activa el evento keyup Ejemplo
mousedown() Adjunta/Activa el evento mousedown Ejemplo
mouseenter() Adjunta/Activa el evento mouseenter Ejemplo
mouseleave() Adjunta/Activa el evento mouseleave Ejemplo
mousemove() Adjunta/Activa el evento mousemove Ejemplo
mouseout() Adjunta/Activa el evento mouseout Ejemplo
mouseover() Adjunta/Activa el evento mouseover Ejemplo
mouseup() Adjunta/Activa el evento mouseup Ejemplo
off() Elimina los controladores de eventos adjuntos con el método on() Ejemplo
on() Adjunta controladores de eventos a los elementos. Ejemplo
one() Agrega uno o más controladores de eventos a los elementos seleccionados. Este controlador solo se puede activar una vez por elemento Ejemplo
$.proxy() Toma una función existente y devuelve una nueva con un contexto particular Ejemplo
ready() Especifica una función para ejecutar cuando el DOM está completamente cargado Ejemplo
resize() Adjunta/Activa el evento de cambio de tamaño Ejemplo
scroll() Adjunta/Activa el evento de desplazamiento Ejemplo
select() Adjunta/Activa el evento seleccionado Ejemplo
submit() Adjunta/Activa el evento de envío Ejemplo
trigger() Activa todos los eventos vinculados a los elementos seleccionados Ejemplo
triggerHandler() Activa todas las funciones vinculadas a un evento específico para los elementos seleccionados Ejemplo
