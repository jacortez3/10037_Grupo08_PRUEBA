SelectoresjQuery

Selectores jQuery

Método / Propiedad Descripción Ejemplos
* Todos los elementos $("*")
#id El elemento con id="apellido" $("#lastname")
.class Todos los elementos con class="intro" $(".intro")
.class,.class Todos los elementos con la clase "intro" o "demo" $(".intro,.demo")
element Todos los elementos < p > $("p")
el1,el2,el3 Todos los elementos < h1 >, < div > y < p > $("h1,div,p")
:first El primer elemento < p > $("p:first")
:last El ultimo elemento < p > $("p:last")
:even Todos los elementos pares < tr > $("tr:even")
:odd Todos los elementos impares < tr > $("tr:odd")
:first-child Todos los elementos < p > que son el primer hijo de su padre $("p:first-child")
:first-of-type Todos los elementos < p > que son el primer elemento < p > de su padre $("p:first-of-type")
:last-child Todos los elementos < p > que son el último hijo de su padre $("p:last-child")
:last-of-type Todos los elementos < p > que son el ultimo elemento < p > de su padre $("p:last-of-type")
:nth-child(n) Todos los elementos < p > que son el segundo hijo de su padre $("p:nth-child(2)")
:nth-last-child(n) Todos los elementos < p > que son el segundo hijo de su padre, contando desde el último hijo $("p:nth-last-child(2)")
:nth-of-type(n) Todos los elementos < p > que son el segundo elemento < p > de su padre $("p:nth-of-type(2)")
:nth-last-of-type(n) Todos los elementos < p > que son el segundo elemento < p > de su padre, contando desde el último hijo $("p:nth-last-of-type(2)")
:only-child Todos los elementos < p > que son el único hijo de su padre $("p:only-child")
:only-of-type Todos los elementos < p > que son el único hijo, de su tipo, de su padre $("p:only-of-type")
parent > child Todos los elementos < p > que son hijos directos de un elemento < div > $("div > p")
parent descendant Todos los elementos < p > que son descendientes de un elemento < div > $("div p")
element + next El elemento < p > que está al lado de cada elemento < div > $("div + p")
element ~ siblings Todos los elementos < p > que aparecen después del elemento < div > $("div ~ p")
:eq(index) El cuarto elemento en una lista (el índice comienza en 0) $("ul li:eq(3)")
:gt(no) Lista de elementos con un índice mayor que 3 $("ul li:gt(3)")
:lt(no) Listar elementos con un índice inferior a 3 $("ul li:lt(3)")
:not(selector) Todos los elementos de entrada que no están vacíos $("input:not(:empty)")
:header Todos los elementos de encabezado < h1 >, < h2 > ... $(":header")
:animated Todos los elementos animados $(":animated")
:focus El elemento que actualmente tiene el foco $(":focus")
:contains(text) Todos los elementos que contienen el texto "Hello" $(":contains('Hello')")
:has(selector) Todos los elementos < div > que tienen un elemento < p > $("div:has(p)")
:empty Todos los elementos que están vacíos $(":empty")
:parent Todos los elementos que son padre de otro elemento. $(":parent")
:hidden Todos los elementos < p > $("p:hidden")
:visible Todas las tablas visibles $("table:visible")
:root El elemento raíz del documento. $(":root")
:lang(language) Todos los elementos < p > con un valor de atributo lang que comience con "de" $("p:lang(de)")
[attribute] Todos los elementos con un atributo href $("[href]")
[attribute=value] Todos los elementos con un valor de atributo href igual a "default.htm" $("[href='default.htm']")
[attribute!=value] Todos los elementos con un valor de atributo href diferente a "default.htm" $("[href!='default.htm']")
[attribute$=value] Todos los elementos con un valor de atributo href que termine en ".jpg" $("[href$='.jpg']")
[attribute|=value] Todos los elementos con un valor de atributo de título igual a 'Tomorrow', o que comiencen con 'Tomorrow' seguido de un guión $("[title|='Tomorrow']")
[attribute^=value] Todos los elementos con un valor de atributo de título que comience con "Tom" $("[title^='Tom']")
[attribute~=value] Todos los elementos con un valor de atributo de título que contenga la palabra específica "hello" $("[title~='hello']")
[attribute*=value] Todos los elementos con un valor de atributo de título que contenga la palabra "hello" $("[title*='hello']")
:input Todos los elementos de entrada $(":input")
:text Todos los elementos de entrada con type="text" $(":text")
:password Todos los elementos de entrada con type="password" $(":password")
:radio Todos los elementos de entrada con type="radio" $(":radio")
:checkbox Todos los elementos de entrada con type="checkbox" $(":checkbox")
:submit Todos los elementos de entrada con type="submit" $(":submit")
:reset Todos los elementos de entrada con type="reset" $(":reset")
:button Todos los elementos de entrada con type="button" $(":button")
:image Todos los elementos de entrada con type="image" $(":image")
:file Todos los elementos de entrada con type="file" $(":file")
:enabled Todos los elementos de entrada habilitados $(":enabled")
:disabled Todos los elementos de entrada deshabilitados $(":disabled")
:selected Todos los elementos de entrada seleccionados $(":selected")
:checked Todos los elementos de entrada marcados $(":checked")
